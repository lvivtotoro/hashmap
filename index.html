How to use Hashmap

Pushing things in

Numbers 0 through 9 push their self into the stack. 10 pushes 1 and 0 into the stack, not 10.
Text between 2 " characters are pushed in as a string.
Characters are pushed in using '. Example: 'a pushes the character a into the stack.

Math

(The following command pop at least one value.)
+ pushes in the sum.
- subtracts the last value from the 2-last.
* multiplies the 2 latest items.
/ divides the last value from the 2-last.
^ Exponent operator.
m is a double-character command:

  • mq pops the latest value and pushes in the square root.
  • mf floors the value
  • mc ceils the value
  • ma pushes the absolute value

Type conversion

s Converts anything into a string.
d Converts a string into a number that the string represents, and converts a character into it's number representation.
c Converts a number into it's character representation or gets the first character of a string.

Operations on strings

s does not only convert anything into a string. If the value is already a string then it needs another command in front of it to perform something on the string:
sl Converts a string to lowercase.
su Converts a string to uppercase.
sr Reverses a string.

Getting input

i Push the input as a string.
h Short for id.

Variables

: Pop a value into a variable.
The variable name is one character (Any character available in the encoding used to read the file).
5:A puts the number 5 into the variable A.
; Pushes a variable into the stack.
;A pushes 5 to the stack, in this case.

Code Blocks

Kind of taken from CJam, a Code Block is basically a function with no name but can be assigned to variables.
Code in between { and } is put in a code block.
Nested code blocks are allowed.
> runs a code block (also pops it)

Outputting

| prints the entire stack, l prints only the latest value.

Clearing the stack

. Pops everything in the stack.
p Pops the latest value.

For loops

A pretty big topic on esoteric programming languages.
A for loop basically repeats a function with at least one different argument until it reaches the end of a list or an iterable.
The first argument is a code block, the second is the object to iterate.
This object can be an array. If it's a string, then the for loop iterates over the characters in the string.
The third argument a character. This character will be the name of a variable only useable in the for-loop.
Example:
{;a|.}i'af Performs a for-loop on the input, puts every object in this iterable in the a variable.
That program prints every character in the input.

While loops

While loops are incredibly giltchy, and I strongly advise not to use them.
The < command performs a while loop. It's argument is a Code Block.
The ! symbol breaks out of a while loop.

Comparing values

The = needs 2 arguments. It is the 2 values to compare.
The command then pushes either a true or a false.

If statements

The ? command runs a code block if a value is true.
The first argument is the boolean and the second one is the the code block.
Example: 22={"yes"}?
That example always prints out "yes" as 2 is equal to 2.
